# 教學：設定 Google 表單並取得參數

為了讓「智簡」網站的聯絡表單資料，能夠**自動存入您的 Google 試算表，並且發送 Email 通知給您**，您需要先完成以下設定：

### 步驟 1：建立 Google 表單
1. 前往 [Google 表單 (Google Forms)](https://forms.google.com/)。
2. 點擊 **「空白」** 建立一個新表單。
3. 將表單標題命名為 `智簡網頁諮詢名單` (或您喜歡的名字)。
4. 新增 **四個簡答題** (請務必依照順序，且都設為「必填」)：
   * 第一題標題：**`公司名稱`**
   * 第二題標題：**`聯絡人姓名`**
   * 第三題標題：**`電子信箱`**
   * 第四題標題：**`聯絡電話`**

### 步驟 2：開啟「接收電子郵件通知」功能
為了讓有人填寫時，您能立刻收到信件通知：
1. 在剛剛建立好的 Google 表單編輯頁面，點擊最上方的 **「回覆」** 標籤。
2. 點擊右側的 **「⋮」** (三個點圖示)。
3. 勾選 **「有新回覆時傳送電子郵件通知給我」**。
   *(這樣一來，只要有人透過網頁填表，您的 Gmail 就會立刻「叮咚」收到通知！)*
4. (可選) 您也可以點擊「連結至試算表」，將收集到的名單自動整理成 Excel 表格。

### 步驟 3：尋找「魔法參數 (Action URL 與 Entry ID)」
接下來需要您提供 4 個藏在表單裡的密碼，我才能幫您把網頁串接起來。您需要在一台電腦上操作：
1. 點擊表單右上角的「眼睛圖示 (預覽)」，打開實際填寫的頁面。
2. 在網頁空白處點擊「右鍵」 ➔ 選擇 **「檢查 (Inspect)」** (打開開發者工具)。
3. 按下 `Ctrl + F` (Windows) 或 `Cmd + F` (Mac) 開啟搜尋框。
4. 搜尋 `<form action`，您會找到一串以 `https://docs.google.com/forms/.../formResponse` 結尾的網址。這就是 **表單的 Action URL**。(請複製整串網址)
5. 接著，搜尋 `entry.`，您會找到三個類似 `name="entry.123456789"` 的代碼。這分別對應您設定的三個問題 (公司名稱、姓名、電話)。(請將這三組數字記下來)
