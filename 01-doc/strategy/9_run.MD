# 智簡 Zhi Jian｜公司營運核心（9_run）

> 依據 Threads @cw.lin_ 提出的「一人式 AI 駐點顧問」新模式，
> 加上 4-googlebusiness 低門檻入口產品線（Google 商家代管）整合為完整漏斗。
> 不賣軟體、不開系統，而是「把一個人 AI 化成一支團隊」，從 2,000 元陪跑到 20 萬月費。

---

## 一、公司成立的「核心」（Why）

### 一句話定位
**中小企業 & 在地店家的「AI 駐點顧問」——從 2,000 元的 Google 商家建檔，一路陪跑到 20 萬月費的深度整合，用 AI 工具、社群代操、快速開發三合一替老闆打一支完整團隊。**

### 核心信念
- 老闆只在乎兩件事：**省錢 / 賺錢**
- 企業需要的不是更多工具，而是「把工具變成戰力」的人
- 大型系統太貴、自由工作者太散、行銷公司太慢 → 智簡是中間的最佳解
- AI agent 不是炫技，是真的用來**產出 3x 效率**、低成本高毛利

### 商業模式骨架｜三段式漏斗

```
┌──────────────────────────────────────────────────────┐
│ L1：Google 商家建檔｜2,000–5,000 NTD（一次性）     │ ← 信任入口・低門檻
│  ↓ 80% 轉換                                           │
│ L2：在地店家月代管｜1,500–3,000 NTD / 月            │ ← 半被動現金流
│  ↓ 30% 升級                                           │
│ L3：智簡標準駐點｜100,000–300,000 NTD / 月          │ ← 高毛利主力
└──────────────────────────────────────────────────────┘
```

| 項目 | L1｜GBP 建檔 | L2｜月代管 | L3｜智簡駐點 |
|---|---|---|---|
| 客單 | NT$ 2,000–5,000 / 次 | NT$ 1,500–3,000 / 月 | NT$ 100,000–300,000 / 月 |
| 目標 | 建立信任、留 GBP 經理人權 | 長期陪跑、養轉介紹 | AI 工具 + 社群 + 開發全包 |
| 同時服務 | 無上限 | 30–50 家 | 3–5 家 |
| 合約 | 一次性 | 月續約 | 3 個月起跳 |
| 成本 | 近乎零 | 單家 < 300 / 月 | 工具總成本 < 3 萬 / 月 |
| 交付 | 2 週內完成建檔 | 月度月報 + LINE 通知 | 每週會議 + LINE 即時 + 月報 |

### 第六個月穩態目標
- L2 在地店家 30 家 × 2,000 = **NT$ 60,000**
- L3 加值（LINE、網站、廣告）10 家 × 3,000 = **NT$ 30,000**
- 智簡駐點 1 家 × 100,000 = **NT$ 100,000**
- **合計約 NT$ 190,000 / 月**（半被動）

## 二、業務（What）

### 三大核心支柱（照抄 Threads 模型）

**① AI 工具導入與團隊賦能**
- 定期盤點、試用、評估最新 AI 工具
- 挑選適用者導入團隊，親自培訓
- 客製化 Prompt 工具包，員工「照抄就能用」
- 內部種子人員培育

**② 社群經營代操（全包）**
- 內容企劃（IG / FB / Threads / LINE 官方帳號）
- AI 文案撰寫（貼文、長文、廣告文）
- AI 圖像製作（素材、封面、海報）
- AI 短影音剪輯（Reels / Shorts / TikTok）

**③ Vibe Coding 快速產出**
- 活動用 Landing Page（3–7 天上線）
- 內部小工具（LINE Bot、表單、查詢機）
- 產品 Prototype / MVP 原型驗證
- 既有網站改版、SEO 優化

### 延伸加值服務
- 流程健檢（免費，獲客入口）
- 自動化工作流串接（n8n / Make / GAS）
- 企業私有知識庫（RAG）
- AI 智慧客服（網站 / LINE）
- 數據洗清 + AI 營運問答
- 一次性 OCR / 合約比對 / 報價單系統

## 三、主要受眾（Who）

> 受眾分**兩大層**：
> **高端層（A–D）** → 智簡駐點（月費 10–30 萬）
> **低門檻層（E–G）** → Google 商家代管（2,000 / 次 + 月 2,000 代管）

---

### 🏢 高端層：智簡 L3 駐點目標客戶

#### 受眾 A：中部中小企業老闆（5–50 人）
- **樣貌**：製造業、貿易業、傳統零售、B2B 服務
- **痛點**：資料混亂、人工作業重、新人難上手、老闆想退但不敢放手
- **決策點**：省人力 / 留住經驗 / 數據看懂

#### 受眾 B：中部品牌主 / 實體店家 / 地方型服務業
- **樣貌**：餐飲、美業、健身、民宿、早午餐、地方農產品（規模較大者）
- **痛點**：產品好但不會說故事、社群沒人顧、自己拍自己剪累爆
- **決策點**：曝光 / 客流 / 品牌感

#### 受眾 C：新創團隊 / 個人品牌 / 自由工作者
- **樣貌**：SaaS 創業者、講師、顧問、教練、KOC
- **痛點**：想快速驗證點子、沒預算請工程師、社群要經營但自己寫不完
- **決策點**：速度 / 成本 / 能 pivot

#### 受眾 D：家族企業二代 / 接班人
- **樣貌**：30–45 歲，接家業但想數位化
- **痛點**：父執輩阻力大、老員工不配合、內部沒技術人
- **決策點**：有懂技術又會溝通的外部盟友

---

### 🏪 低門檻層：Google 商家代管目標客戶（參考 `set_upgrade.MD`）

#### 受眾 E｜Tier 1：在地實體小店老闆（核心 70%）
- **樣貌**：餐飲、機車行、小賣店、髮廊、診所、寵物店、補習班
- **地點**：學校 / 商圈 / 夜市周邊 1km，能走路到的店
- **規模**：1–5 人小店，老闆自己顧店，35–60 歲
- **痛點**：不會 GBP 後台、照片少、評論沒回覆、隔壁店生意好不知為什麼
- **預算**：月 2,000–3,000 願付；月 10,000+ 要說服
- **篩選標準**：評分 3.8–4.5、照片 < 5 張、評論 < 30 則、3 個月無更新

#### 受眾 F｜Tier 2：已合作半年的 L2 月代管店家（20%）
- **來源**：L1 建檔後升級月代管，做滿 6 個月建立信任
- **可升級服務**：一頁式網站（5,000 起）、LINE 官方帳號（3,000）、季度攝影（2,000）、在地 Google Ads（5,000 / 月起）
- **價值**：真正的長期金礦，回購 + 加值 + 轉介紹三軸收入

#### 受眾 G｜Tier 3：跨界轉介的中型企業（10%）
- **來源**：小店老闆口耳相傳「有個會用 AI 幫店家弄網路的」
- **需求**：連鎖店 GBP 整體管理（10 家分店以上）、中小企業內部 AI 工具、扶輪社 / 商會介紹的數位轉型需求
- **流向**：直接轉進智簡 L3 駐點（10–30 萬 / 月）

## 五、立即行動的獲客路徑

### 高端層（L3 智簡駐點）
1. **Threads / IG 內容經營**：每週 3 則，秀「一人接 3 家」的實戰案例 + Demo 截圖
2. **免費流程健檢**（30 分鐘）：官網表單 → 現場 Demo → 報價
3. **Demo Hub 展示**：OCR / 合約比對 / AI 客服，讓老闆秒懂能解什麼痛
4. **地方社群滲透**：中部商會、青商、扶輪、二代接班社團
5. **案例內容行銷**：每完成一個案子，產出一篇案例文 + 一支短影音

### 低門檻層（L1 / L2 在地店家）
6. **走街拜訪**：從 Google 地圖挑 100 家符合篩選條件的店，一週走訪 25 家
7. **話術破冰**：「隔壁那間 XX 評論一直跳，您這邊客人一定被搶走不少」
8. **現場展示**：當場打開店家 GBP，指出照片、評論、貼文三個漏洞
9. **免費健檢報告**：10 分鐘整理出 5 個可修正點，建立第一次信任
10. **L1 → L2 → L3 漏斗經營**：2,000 開始養，半年後升加值，一年後轉大單

## 六、未來 12 個月里程碑建議

| 月份 | L1 / L2 目標（低門檻） | L3 目標（高端駐點） |
|---|---|---|
| M1–M2 | 成交 5 家 L1 建檔，驗證話術 | 鎖定第 1 家月費客戶、產出 3 支案例影片 |
| M3–M4 | L2 月代管達 10 家，月金流 2 萬 | 同時服務 2 家駐點，建立 SOP 與工具包 |
| M5–M6 | L2 達 20 家，首批 L3 加值（LINE / 網站）3 家 | 第 3 家駐點上線，月費端達 30 萬 |
| M7–M9 | L2 達 30 家、L3 加值 7 家，月金流 9 萬 | 穩定接案 4 家駐點，社群追蹤破 3,000 |
| M10–M12 | 半被動收入達 19 萬 / 月 | 年度駐點營收突破 200 萬 |

**年度合計目標**：**~ 300 萬 NTD / 年**（L2 + L3 加值 ~100 萬 + L3 駐點 ~200 萬）
