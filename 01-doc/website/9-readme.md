### 📍 第一區塊：Hero（首頁第一眼）——「一人顧問 = 一支團隊」的承諾

**【心理學目的】**：第一秒告訴老闆「你不用再養一支團隊，我就是你的團隊」。

- **標題**：別再養一支永遠缺人的團隊。
- **副標（金色 Hook）**：一位顧問、一個月費，把 AI 變成你真正的戰力。
- **描述**：智簡是您的 **AI 駐點顧問**。我們幫企業做三件事：**導入 AI 工具並培訓團隊、代操社群內容產製、快速開發 Landing Page 與小工具**。一個人抵一支團隊，月費制長期陪跑，中小企業也請得起。
- **CTA**：👉 免費預約 30 分鐘流程健檢（現場告訴您一個月能省多少人力）
- **信任指標**：50+ 合作企業 ｜ 月費 10 萬起 ｜ 首次諮詢免費

### 📍 第二區塊：痛點放大（The Agitation）

**【心理學目的】**：戳中「請不起團隊、又不想等死」的中間夾層痛苦。

**「您的公司，是不是也卡在這四種痛裡？」**

1. **人力請不起**：想要社群經營、想做網站、想串自動化，但請一個專才就要 5 萬起跳。
2. **工具追不完**：ChatGPT、Claude、Gemini、Midjourney… 新工具一週一個，老闆根本不知道該買哪個、怎麼用。
3. **內容產不出**：知道該經營社群，但自己寫自己拍根本沒時間，外包行銷公司又貴又慢。
4. **系統做不動**：想做一個小工具、一個活動頁、一個機器人，找工程師要一個月起跳，報價還三十萬。

### 📍 第三區塊：三大核心支柱（The Solution Pillars）

**【心理學目的】**：用三支柱清楚定位，讓老闆明白「這個人把我需要的三種人合成一個」。

**「我們不只是導入工具，我們是您真正的數位營運夥伴。」**

#### 🧠 支柱一：AI 工具導入與團隊賦能
> **替您的團隊裝上 AI 大腦**
- 每月盤點、評估、篩選最新 AI 工具
- 親自帶領團隊使用、產出 Prompt 工具包
- 培育內部種子人員，確保我們離開後團隊仍然強大
- **核心獲益**：員工產出 3× 效率，學習成本趨近於零

#### 📣 支柱二：社群經營代操
> **一個人完成企劃、文案、圖像、短影音**
- IG / FB / Threads / LINE 官方帳號內容全包
- AI 文案 + AI 圖像 + AI 短影音剪輯
- 每月固定產出量 + 廣告文案優化
- **核心獲益**：不用請小編、不用請剪輯、不用請設計，一個月費全部搞定

#### ⚡ 支柱三：Vibe Coding 快速開發
> **3–7 天交付你想要的數位工具**
- Landing Page / 活動頁 / 產品頁
- LINE Bot / 查詢工具 / 極簡 ERP / 預約系統
- MVP / Prototype 原型驗證
- **核心獲益**：不用找工程師、不用排三個月，想到明天就開始做
